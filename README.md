# Cambridge Hackathon 2018 - Last minute essay assistant

Via connections to Microsoft Azure ML APIs contributes a number of helpful functionalities to the essay writing process.


## Features
+ Word Count tracker with text editor
+ Bing Image Search
+ Search for relevant sections of scholar articles
+ Progress tracker, is the paper going to be completed before the deadline?
+ Face tracker to ensure attention is kept on the page and breaks suggested when distress emotion detected.


## Tech Stack
Developed using angular Frontend.
Uses variety Microsfot Azure ML Apis.
